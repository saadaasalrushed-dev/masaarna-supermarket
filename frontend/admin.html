Backend API connection

Products and categories load from the same SQLite database as the storefront. If those tabs show “Not Found”, the browser is calling the wrong server — set the URL where node server.js is running.

On localhost, this page auto-detects the API on ports 3000–3015 when this origin has no Masaarna /api/health (e.g. admin opened on 3003, API on 3000).
